*# 📊 Business Analytics Dashboard (Excel)

## 📌 Overview

This project demonstrates the creation of an interactive Business Analytics Dashboard using Microsoft Excel. The dashboard transforms raw business data into meaningful insights using Pivot Tables, Pivot Charts, Slicers, formulas, and data visualization techniques.

The project includes dashboards for Customer Service, Finance, and Order Analysis, allowing users to explore business performance through interactive filters and charts.

---

## 🚀 Features

- 📈 Interactive Excel Dashboards
- 📊 Pivot Tables & Pivot Charts
- 🎯 KPI Analysis
- 🎛️ Slicers for Dynamic Filtering
- 📉 Customer Service Analysis
